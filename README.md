# Projeto linux-system-monitor.

  Este projeto é um script simples em Bash criado para monitorar os principais recursos do sistema linux
  diretamente pelo terminal.
  
  Aferramenta permite verificar rapidamente o desempenho do sistema, até em tempo real
  sem precisar instalar programas adicionais .

  FUNCIONALIDADES:
  Monitoramento de carga da CPU em tempo real .
  Exibição  total da memoria RAM em MB e quanto esta em uso .
  Exibição total do disco e quanto em uso .
  Alerta quando o sistema estiver com carga alta .

  INSTALAÇÃO 

  1) Acesse a pasta scripts :
     cd scripts 
  2)Excute :
    bash install.sh
  OBS: Após isso o comando estara disponivel em sistema.

 MODO DE USO 
 
 Execute o comando abaixo em qualquer parte do terminal:
  dlinux
 
 OBS: O script começará a mostrar as informações do sistema em tempo real no terminal
